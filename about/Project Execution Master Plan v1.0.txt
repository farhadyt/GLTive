GLTive Platform
Project Execution Master Plan v1.0
1. Project məqsədi

GLTive modul əsaslı, çoxdilli, on-prem deploy oluna bilən, AI dəstəkli enterprise IT operations platformasıdır. Platforma asset, stock, ticketing, infrastructure, network, monitoring, security, automation, backup orchestration və AI-assisted operations capability-lərini bir sistemdə birləşdirəcək.

2. Əsas memarlıq prinsipi

Platforma genişlənə bilən modular skeleton ilə qurulmalıdır.
Heç bir modul sonradan əlavə ediləndə əsas skeleti dağıtmamalıdır.

3. Əsas development qaydası

Development modul-modul aparılacaq.
Hər modul:

ayrıca bounded context (məhdud məsuliyyət sahəsi)
ayrıca service layer
ayrıca database ownership qaydası
localization-ready UI
audit-ready state changes
ilə qurulmalıdır.
4. İlk gündən məcburi foundation qaydaları

Aşağıdakılar bütün modullarda ilk gündən olmalıdır:

multi-language support
role/permission readiness
audit logging readiness
tenant/company separation readiness
notification readiness
API-first internal design
future module activation readiness
5. Çoxdillilik prinsipi

Platforma ilk gündən i18n-ready (çoxdilli infrastruktur hazır) qurulmalıdır.

Başlanğıc dillər:

Azerbaijani
Russian
English
Turkish
Arabic

Qaydalar:

UI string-lər hardcoded yazılmamalıdır
bütün mətnlər translation key ilə idarə olunmalıdır
notification template-lər də localization-ready olmalıdır
tarix, saat, rəqəm və format locale-aware olmalıdır
Arabic üçün RTL-readiness (sağdan-sola dizayn hazırlığı) nəzərə alınmalıdır
6. Development strategiyası

İlk mərhələdə bütün platforma birdən yazılmayacaq.
Əvvəl foundation və ilk əsas modul qurulacaq, sonra növbəti modullar əlavə olunacaq.

7. İlk development modulu

Module 01: Stock / Inventory Foundation

Bu modul platformanın asset və lifecycle hissəsinin təməlini təşkil edəcək.

İlk scope:

stock items
item categories
item models
brands/vendors reference
warehouse/location relation
quantity tracking
stock in / stock out
stock movement history
serial/no-serial support
minimum stock level
audit trail
multi-language UI foundation

Bu modul gələcəkdə aşağıdakılar üçün baza olacaq:

asset registration
assignment/handover
procurement
maintenance
disposal
QR/barcode
reporting
